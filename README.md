# GoldenTicket_Android

당일 공연 추첨식 예매 서비스 **골든 티켓**입니다.

------------
![GoldenTicketIcon](https://user-images.githubusercontent.com/49272528/61025745-8f1a8500-a3ec-11e9-8e5a-ae246f2a9bbc.png)

어플 기능
------------
> * 로그인
> * 회원가입
> * 오늘 공연 보여주기
> * 공연 검색하기
> * 공연 응모하기
> * 응모한 공연 타이머 보기
> * 응모한 공연 당첨확인
> * 응모한 공연 당첨확인 알람
> * 매일 오전 10시 마다 오늘의 공연 알람
> * 관심있는 공연 알람
